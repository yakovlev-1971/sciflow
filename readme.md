# 🧬 SciFlow — Мониторинг научных новостей

**SciFlow** — это облачный сервис для автоматического сбора, группировки и доставки научных новостей. Он отслеживает публикации из ведущих мировых источников и присылает свежие заголовки прямо в Telegram.

## 📡 Источники

SciFlow собирает новости из **47 источников**, включая:

- **Nature** (6 журналов, включая Communications, Climate Change, Genetics)
- **Science.org**, **PNAS**, **arXiv**
- **Science News**, **ScienceDaily**, **EurekAlert**, **Phys.org**
- **BBC Science**, **NYT Science**, **The Guardian Science**
- **NASA**, **ESA**, **CERN**, **WHO**, **CDC**
- **MIT**, **Stanford**, **Harvard**, **Caltech**, **Oxford**, **Cambridge**
- Российские: **N+1**, **Naked Science**, **Коммерсант Наука**, **РБК**, **Lenta.ru**

## 🔧 Возможности

- **Автоматический мониторинг** — сбор новостей каждые 10 минут
- **Telegram-уведомления** — новые заголовки приходят прямо в мессенджер
- **Веб-дашборд** — удобный интерфейс для просмотра ленты в браузере
- **Группировка по источникам** — новости сортируются по изданиям
- **Защита паролем** — доступ только для авторизованных пользователей

🔍 Группировка дубликатов (эмбеддинги)
SciFlow умеет находить семантические дубликаты — когда одна и та же новость выходит на Phys.org, ScienceDaily и EurekAlert. Используется модель all-MiniLM-L6-v2 и библиотека sentence-transformers. Работает, но добавляет зависимости и замедляет парсинг. Требуется оптимизация или вынос в отдельный модуль.

📨 Уведомления только о новых новостях
Сейчас SciFlow отправляет в Telegram все заголовки при каждом обновлении. Прототип с кэшированием (сравнение с предыдущим прогоном через json) уже проверен на RegionFlow. Будет перенесён в SciFlow после финальной отладки.

🌍 Региональная версия (RegionFlow)
На базе SciFlow сделан отдельный проект для мониторинга региональной повестки — RegionFlow (Орловская область). Работает на Yandex Cloud, включает RSS-источники и парсинг Telegram-каналов. Кодовая база совместима, планируется унификация.

📡 Парсинг Telegram-каналов
Публичные каналы парсятся через веб-версию t.me/s/ без API и токенов. Метод проверен на RegionFlow, но требует аккуратного обращения с блокировками и капчами.

🧹 Парсинг сайтов без RSS
Для источников, у которых нет нормальной RSS-ленты (например, orel-region.ru), написан HTML-парсер на BeautifulSoup. Работает, но зависит от структуры сайта и требует периодической поддержки.

🖥 Деплой на Yandex Cloud
SciFlow и RegionFlow протестированы на виртуальной машине Yandex Cloud. Деплой работает, но требует ручной настройки SSH, Git и systemd для автозапуска. Планируется автоматизация через скрипты развёртывания.
